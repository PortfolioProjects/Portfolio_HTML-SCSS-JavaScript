Cafe Orders - Angular, Express, Mongo

← Back

Cafe Orders is an application created with Express, MongoDB, Angular6 and materialize-css.

Cafe Orders is for waitresses and Cafe-owners - to add products to an order and finalize it, to add new products or categories in the menu, to display analytics.

Languages, Frameworks

Angular 6, Express,

mongoose: MongoDB database ODM

Used Libraries

jwt, bcryptjs, moment, multer, validator

materialize-css, ngx-toastr, chart.js

General functionality

Authenticate users via JWT

Not authenticated user can: register, login

Every authenticated user can:

  • create category and product
  • upload category image
  • update/delete category or product
  • add products to order

General functionality

Every authenticated user can:

  • remove a product from total order
  • finish order and display it in history
  • search in history for given dates
  • display statistics for orders
  • display overview of yesterday's orders

App can display success and error messages
