Dream Rest - Express, MongoDB, Pug

← Back

This Web Application is developed by Team "PerfectCodeStorm" as a part of the "Web Apps with Node.js" course at Telerik Academy 09/2017.

Dream Rest is an app to book rooms and services in guest houses within the beautiful Bulgarian mountains.

Libraries, Frameworks

Express, Pug - server-side rendering

MongoDB - database, connect-mongo,

Used Libraries

passport, passport-local, connect-flash
Selenium, Mocha, Sinon, Supertest, Istanbul

Main features

Authenticate users: register, login, logout
User roles - Admin and User
Admin can:
create new regions, new hotels in a region
add rooms and services for a hotel
update region, hotel, room, service
Admin has admin-panel to control all data

Main features

Authenticated user can:
book a room or service
display hotel, room and service details
User's profile shows user info and all bookings
Non authenticated user can:
display a list of regions or hotels
display regions' details
