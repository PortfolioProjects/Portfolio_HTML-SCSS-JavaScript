Dream Rest - Express, MongoDB, Pug

← Back

Web Application developed by Team "PerfectCodeStorm" as part of the Web Apps with Node.js course at Telerik Academy 09.2017.

Dream Rest is an application to book rooms and services from guest houses in beautiful Bulgarian mountains.

Libraries, Frameworks

Express, Pug - server-side rendering

MongoDB - database, connect-mongo,

passport, passport-local, connect-flash

Testing - Selenium, Mocha, Sinon, Supertest, Istanbul

Main features

  • Authenticate users: register, login, logout
  • User roles - Admin and User
  • Admin can:
    • create new regions, new hotels in a region
    • add rooms and services for a hotel
    • update region, hotel, room, service
  • Admin has admin-panel to manipulate all data

Main features

  • Authenticated user can:
    • book room or service
    • display hotel, room and service details
  • User's profile shows user info and all bookings
  • Non authenticated user can:
    • display list of regions, list of hotels
    • display region-details: all hotels in the region
