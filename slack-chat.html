Slack Chat - React, Redux, Firebase

← Back

This is a Chat application similar to the Slack-app. For the app creation I use React, Redux, and Firebase - to store data and images, and to host the finished app.

In ChatApp_React each user can create a profile, join channels and create private chats, personalize main colors of the page.

Frameworks, Libraries

React - library for building user interfaces
Redux - state container for JavaScript apps
Firebase - database, Storage, Hosting

Other Used Libraries

semantic-ui-react, react-avatar-editor
moment, emoji-mart, react-color

General functionality

Authentication (register, login, logout)
Every authenticated user can:
create a new channel
star the current channel
send text or image messages
add emoji to text messages

General functionality

Every authenticated user can:
create private Direct Messages with another user
change main colors of the page
search messages in the channel by username
change avatar - upload an image and edit it
Not authenticated users can register or login
