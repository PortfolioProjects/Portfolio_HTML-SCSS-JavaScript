The Venue - React

← Back

An application informing the user about a venue (hall) and its future events.

TheVenue is created with React, back-end and database can be added later.

Libraries, Frameworks

React - library for building user interfaces

material-ui/core, material-ui/icons

react-reveal, react-scroll, react-slick

General functionality

  • Featured Component - with Carousel and TimeUntil
  • Carousel:
    • used react-slick/Slider
  • TimeUntil:
    • used react-reveal/Fade to animate time
  • VenueNfo:
    • uses react-reveal/Zoom to zoom info elements

General functionality

  • Highlights - with Description and Discount
  • Discount:
    • uses Fade, Slide from react-reveal
    • uses reusable component MyButton
  • Pricing:
    • uses react-reveal/Zoom and MyButton
  • Location:
    • uses google maps iframe to show the location
